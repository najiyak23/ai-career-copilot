# AI Career Copilot

An AI-powered career assistant built with Streamlit and Google Gemini.

## Features
- Upload Resume PDF
- Paste Job Description
- Resume Q&A
- Resume vs Job Match Score
- Personalized Interview Questions

## Tech Stack
- Python
- Streamlit
- Google Gemini API
- PyPDF2
